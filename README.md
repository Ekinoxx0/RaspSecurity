# RaspSecurityTest

This project is an implementation of a security system based on a Raspberry Pi linked to sensors and buttons.

Note:
 - RedLed : 24
 - BlueLed : 25
 - YellowLed : 27
 - GreenLed : 28
 
 - Buzzer : 15
 
 - c : 23, 24, 25, 27
 - l : 0, 1, 2, 3
 
 - alarm : 16
 - Opening : 7
 - Move : 4
